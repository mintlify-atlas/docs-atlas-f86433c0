## What's Included

All Birdflop hosting plans come with:

- **3 off-site backups** for data security
- **DDoS protection** to keep your server online
- **Dedicated IPs** on 8+ GB plans
- **Enhanced Pterodactyl Panel** for server management
- **3-day satisfaction guarantee** for peace of mind
- **Unmetered NVMe storage** on premium plans
- **Multi-core CPU allocation** for optimal performance

<Note>
  As a nonprofit organization, all revenue is used to cover server costs, build new infrastructure, and develop free public resources for the community.
</Note>

## Where Your Payments Go

As a 501(c)(3) nonprofit:

1. **Server Costs**: Building new servers, colocation fees, and hardware
2. **Software Licensing**: Control panel and infrastructure software
3. **Free Resources**: Development of public tools and resources
4. **Client Reimbursements**: Excess profit returned to active clients
