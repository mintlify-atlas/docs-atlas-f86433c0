## Overview

BirdTickets is a Discord ticket bot which provides premium features without a premium cost. It offers a complete ticketing solution for Discord servers with advanced features typically found only in paid bots.

## Getting Started

### Invite BirdTickets

Add BirdTickets to your Discord server:

<Card title="Invite BirdTickets" icon="discord" href="https://discord.com/oauth2/authorize?client_id=809975422640717845&permissions=0&scope=bot">
  Click to invite BirdTickets to your Discord server
</Card>

### Setup Instructions

1. **Invite the Bot**: Use the invite link above to add BirdTickets to your server
2. **Configure Permissions**: Ensure the bot has appropriate permissions to create channels and manage messages
3. **Set Up Ticket Category**: Create or designate a category for ticket channels
4. **Configure Settings**: Use bot commands to customize ticket behavior, auto-close timers, and transcript settings
5. **Create Ticket Panel**: Set up a message with reactions or use commands to allow users to create tickets

## Key Features

### Ticket Creation

- Command-based ticket creation
- Reaction-based ticket creation from custom panels
- Multiple ticket categories for different support types

### Ticket Management

- Close tickets via commands or reactions
- Automatic closure when users leave the server
- Inactivity timer for abandoned tickets
- Transcript generation for all tickets

### Transcripts

- Beautiful HTML-formatted transcripts
- Automatic delivery to ticket creators
- Optional logging to designated staff channels
- Permanent record of all support interactions
